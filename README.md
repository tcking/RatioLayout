# RatioLayout
Layout's width/height with specified ratio,for example:

``` xml
<com.github.tcking.view.RatioRelativeLayout
        android:layout_width="match_parent"
        android:layout_height="wrap_content"
        app:ratio="4:3"
        >
        ...
 </com.github.tcking.view.RatioRelativeLayout>
```
